Rust

Rust es un lenguaje de programación moderno de sistemas, compilado y multiparadigma, enfocado en ofrecer alto rendimiento, seguridad de memoria y concurrencia sin recolector de basura, previniendo errores comunes de C/C++ mediante su sistema de propiedad (ownership) y préstamos (borrowing). Se utiliza ampliamente en el desarrollo de sistemas operativos, motores de juegos, navegadores y software crítico, ya que permite escribir código eficiente, seguro y confiable sin sacrificar control sobre el hardware.
