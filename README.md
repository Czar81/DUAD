# DUAD - Exercise Directory
This directory contains all the exercises done, organized by modules, weeks and topics.
## Directory structure
Each module contains its own weeks, and the weeks have their topics and README file.
The structure is:
- *Modules*
    - *Weeks*
        - *README file*
        - *Topics*
            - *Exercises*
